Roadmap

These are the major features we're planning to implement on the road to the Torsion 2 release. Note that these plans can change and are not a guarantee of future releases and features.

  • Editor position tracking.
  • Split editor windows.
  • Refactoring tools.
  • New docking window system.
  • Additional syntax highlighting customization.
  • More improvements to Torque's TelnetDebugger.
  • Opening of non-script files in editor.
  • Unicode support.
  • Live project tree icons.
  • Further TorqueScript language enhancements.

If you have any other feature requests or suggestions please post them to our bug tracking system for consideration.
