What is Torsion?

Torsion is an integrated development environment specifically for creating and debugging TorqueScript based games and mods. It does not edit C++ source, have a WYSIWYG GUI editor, or work with TorqueX C# code... Torsion is solely focused on C++ Torque engine development using TorqueScript. Torsion is a complement to the existing tools and editors that ship with the various Torque C++ engines.

Getting Started

If you want to jump right in and your new to Torsion you should start with the Create a New Project and Using the Debugger walk thrus. If you've upgraded from a previous version of Torsion you should look at the change log to see what new features we've added and bugs we've fixed.
