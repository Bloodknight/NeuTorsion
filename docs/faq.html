FAQ

Q: It seems the watch window doesn't evaluate variables properly... is this a bug?

A: Yes it is a bug. In particular it was a bug in the TelnetDebugger that seems to be fixed for TGE and TGB, but not for TGEA. There is a bug fix posted here.


Q: Why can i not step thru code or set breakpoints on code outside of functions?

A: By default the Torque engines disable this feature for performance. Although it is arguable if it is much of a win at all, the extra breaklines is not enabled in release builds by default. Open engine\console\astNodes.cc/cpp and enable the TORQUE_EXTRA_BREAKLINES define and rebuild your executable.


Q: When i launch my game using Torsion nothing happens... what is wrong?

A: Most likely you've tried to run your game from inside a mod folder. Your project working directory should contain the subdirectory where you game scripts are and be the same folder where the Torque executable and entry script exist.
